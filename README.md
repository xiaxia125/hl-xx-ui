# hl-select-table

## API

### Attributes

| Name                   | Description                     | Type                       | Default |
| --------               | ------------------------------- | -------------------------- | ------- |
| model-value / v-model  | 绑定值，返回勾选的数据集合（可跨页）         | Array                      | [ ]      |
| columns                | 表格列的配置描述，具体项见后文     | Array                      | [ ]      |
